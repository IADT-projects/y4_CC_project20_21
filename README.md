# # project_y4 repo


The starter repository for the DL836 y4 project.
Upload your documentation and the presentation to the docs folder.Please note:




## Source Code
Upload your source the code for your project to the "Source" folder

## Report
Upload your working documents to the "writing" folder. You also have to submit the final version using TurnItin document.

## Screencast
Create a walkthrough of your final project with a voice-over. Upload a .mp4 file to video. Observe 100 Mb file limitation!

## Website
The 'docs' folder contains a stub HTML file that you can use as a project blog. You can either use the included index.html file (rename it by removing the '_') or use the provided jekyll template which uses markdown to render html.
